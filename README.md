# CRON Expression Parser

## Running

Install dependencies:

```bash
npm install
```

Run:

```bash
npm run start "{input}"
```

Run tests:

```bash
npm run test # run all tests (unit and acceptance)

npm run test-unit # run just the unit tests

npm run test-acceptance # run just the unit tests
```
